SPREMENLJIVKE

En del pomnilnika namenjen nasemu programu (procesu)
Ta del je razdeljen na 4 dele
4. DELI
stack (sklad), (lokalne/avtomatske spremenljivke so doma na skladu)
heap (kopica)    
data (tukaj se nahajajo spremenljivke)
code (del pomnilnika iz katerega pomnilik jemle ukaze in jih izvrsuje (nasa koda))

vrste spremenljivk;
>extern (z extern lahko deklariramo samo staticne podatke)
>static -> (stat. podatki), zivijo v data
>(auto) = lokalne (zivi dokler se funkcija izvaja) zivi v skladu (se ne uporablja)
>register (za hitrejso shrambo spremenljivk, se ne uporablja)

Ko se funkcija izvede se sklad sprosti (sklad se premika gor in dol)
